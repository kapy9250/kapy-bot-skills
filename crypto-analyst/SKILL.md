## 分析框架

### 1. 市场概况 (Dashboard)

| 维度 | 数据点 | 解读方法 |
|------|--------|----------|
| 价格 | price, change_24h | 趋势方向 |
| 均线 | ma50, ma200 | 牛熊分界 |
| RSI | rsi (0-100) | <30超卖, >70超买 |
| 情绪 | fear_greed | <25极度恐惧, >75极度贪婪 |

### 2. 多维度分析矩阵

**技术面分析**:
```
趋势判断:
- 价格 vs MA50: 短期趋势
- 价格 vs MA200: 长期趋势 (牛熊分界)
- RSI: 超买超卖
- MACD: 动能方向

关键公式:
- MA200距离% = (price / ma200 - 1) * 100
- RSI极端值: <20或>80 历史罕见，反转概率高
```

**情绪面分析**:
```
Fear & Greed 解读:
- 0-25: 极度恐惧 → 逆向做多信号
- 25-45: 恐惧 → 谨慎偏多
- 45-55: 中性 → 观望
- 55-75: 贪婪 → 谨慎偏空
- 75-100: 极度贪婪 → 逆向做空信号

连续天数:
- streak.days > 5: 情绪可能即将反转
- streak.days > 10: 强烈反转信号
```

**衍生品分析**:
```
Funding Rate:
- > +0.05%: 多头过热，回调风险
- < -0.02%: 空头拥挤，轧空风险
- 接近0: 市场平衡

多空比:
- long_ratio > 70%: 多头过度拥挤
- long_ratio < 40%: 空头主导

OI (持仓量):
- OI上升 + 价格上升 = 趋势健康
- OI上升 + 价格下降 = 空头加仓
- OI下降 + 价格下降 = 多头投降
```

**ETF/机构流向**:
```
流向解读:
- 连续流入 > 3天: 机构看多
- 连续流出 > 3天: 机构减仓
- 恐慌中流入: 聪明钱抄底

关注点:
- IBIT (BlackRock): 机构风向标
- GBTC: 套利和赎回压力
```

**挖矿数据**:
```
算力趋势:
- 算力上升: 矿工看好后市，网络健康
- 算力下降: 矿工关机或抛售，利润承压

矿工投降信号:
- 算力 30d 下降 > 10%
- 同时价格创新低
- 历史上往往是底部区域
```

**宏观环境**:
```
利率环境:
- 10Y上升: 风险资产承压
- 10Y-2Y利差:
  - 正值: 经济预期正常
  - 负值(倒挂): 衰退预期

流动性:
- Fed RRP下降: 流动性改善，利好风险资产
- Fed RRP上升: 流动性收紧
```

### 3. 综合评分系统

| 维度 | 权重 | 评分规则 |
|------|------|----------|
| 技术面 | 30% | RSI + 均线 + MACD |
| 情绪面 | 20% | F&G + 连续天数 |
| 衍生品 | 20% | Funding + 多空比 |
| 机构流向 | 15% | ETF 流向趋势 |
| 宏观 | 15% | 利率 + 流动性 |

**评分转换**:
```
-100 ~ -60: 极度看空 🔴🔴
-60 ~ -20:  看空 🔴
-20 ~ +20:  中性 ⚪
+20 ~ +60:  看多 🟢
+60 ~ +100: 极度看多 🟢🟢
```

为每次分析提供三种情景：

| 情景 | 概率 | 条件 | 目标价 |
|------|------|------|--------|
| 牛市/反弹 | X% | 触发条件 | $XXX |
| 震荡/横盘 | Y% | 触发条件 | 区间 |

### 5. 操作建议

**按交易周期分类**:

```
短线 (1-7天):
- 入场点、止损、止盈
- 仓位建议 (占总资金%)
- 风险收益比

波段 (2-8周):
- 趋势确认信号
- 加仓/减仓策略
- 关键价位

长线 (3-12个月):
- 定投建议
- 目标价位
- 持仓比例
```
